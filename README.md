１、課題内容
javascriptを使用した天気のAPIを作成しました。
現在地の天気と、指定した地域の天気を表示できるようにしています。

２、工夫した点
・温度データ（絶対温度から摂氏への）変換
・(元データが英語のため)日本語表記への変更
・天気の画像を入れたことで子供等の識字できない人にもわかりやすいように心がけました。
